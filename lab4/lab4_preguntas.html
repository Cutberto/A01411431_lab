Preguntas del laboratorio.

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje compilado de alto nivel y altamente tipado. Este es un lenguaje pensado para el diseño de aplicaciones robustas, y es ampliamente utilizado para desarrollar aplicaciones de escritorio, además de ser el lenguaje nativo de las aplicaciones Android. Por otra parte, javascript es un lenguaje interpretado de alto nivel, débilmente tipado. Este lenguaje está orientado al desarrollo web al ejecutarse mediante navegadores, además de ser orientado al desarrollo frontend.
